◆ PrintName()

void ArgPair_base::PrintName ( int  width = 0)
protected

Method that defines the first part of the derived ArgPair::Print method.

Since the behavior is the same for each type of parameter, this method has been defined in the ArgPair_base base clase.

Parameters
widthUse this parameter if you want to print the name of the parameter in a certain column width. This can be useful when printing a table using PrintAll.

An indent is printed by default.
